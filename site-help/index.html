# 添加南科手册到桌面

南科手册支持PWA(Progressive Web App)，这意味着您可以将它添加到桌面，并作为一个本地应用使用。因此，在下载应用后，即使在网络不佳的环境下，您依然可以阅读南科手册上的内容。

# 如何添加

以Chrome为例：

  1. 点击右上角的三个点

  1. 点击“添加到主屏幕”

  1. 点击“添加”
